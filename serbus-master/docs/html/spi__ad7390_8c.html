Detailed Description

Uses serbus to control an AD7390 DAC.

Author
Alex Hiam - alex@.nosp@m.gray.nosp@m.cat.i.nosp@m.o

Requires an SPI Kernel driver be loaded to expose a /dev/spidevX.Y interface and an AD7390 be connected on the SPI bus.
