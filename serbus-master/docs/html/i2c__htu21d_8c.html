Detailed Description

Uses serbus to get RH and temperature data from an HTU21D.

Author
Alex Hiam - alex@.nosp@m.gray.nosp@m.cat.i.nosp@m.o

Requires an I2C Kernel driver be loaded to expose a /dev/i2c-N interface and an HTU21D be connected on the I2C bus.
